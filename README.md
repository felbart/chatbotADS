# 🍔 ChatBot TopBurg — Projeto Integrador I-A

Chatbot de atendimento ao cliente desenvolvido em Java como projeto acadêmico da disciplina **Projeto Integrador I-A** do curso de Análise e Desenvolvimento de Sistemas da **PUC Goiás (EAD)**.

## Sobre o projeto

O programa simula o atendimento digital de uma lanchonete fictícia chamada **TopBurg**, interagindo com o usuário via terminal por meio de menus numerados com respostas fechadas.

O desenvolvimento aplicou conceitos de:
- Tipos de dados e variáveis
- Entrada e saída de dados
- Estruturas de seleção (`if/else` e `switch/case`)
- Estruturas de repetição (`do/while`)

## Como executar

**Pré-requisito:** JDK instalado

```bash
# Compilar
javac Main.java

# Executar
java Main
```
